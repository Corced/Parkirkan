# Parkirkan - Park it

## What is Parkirkan?
Parkirkan is a parking management system that helps parking owners manage their parking lots and transactions.

## Features
- Parking lot management
- Transaction management
- User management
- Role-based access control
- Dashboard
- Reporting

## Tech Stack
- Laravel
- Next.js
- MySQL
- Tailwind CSS
- Chart.js
- Lucide React

## Installation

### Backend
```bash
cd code/backend
composer install
php artisan key:generate
php artisan migrate
php artisan serve
```

### Frontend
```bash
cd code/frontend
npm install
npm run dev
```
